This is CA Monitor version 3.0-SNAPSHOT
Description: This service checks whether the CA API gateway contains new or changed webservices.
Developed by: Team Applicatie Integratie (Gemeente Den Haag)
Website: https://werknet.denhaag.nl/Content/overzichtspagina/Paginas/Webservices.aspx

To install the CA Monitor service, you need to execute bin/install.cmd as administrator.
This script installs the service.

To uninstall the CA Monitor service, you need to execute bin/uninstall.cmd as administrator.
